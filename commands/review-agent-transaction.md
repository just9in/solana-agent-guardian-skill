# Command: review-agent-transaction

Review a Solana transaction generated by an AI agent before signing.

Accept base64 transaction, decoded JSON, simulation result, policy manifest, or user intent. Load transaction risk scoring, simulation, and policy files. If decode/simulation is unavailable, return `NEEDS_MORE_DATA`, not a false approval.

Output exactly: Decision, Confidence, Reason, Policy rules matched, Programs observed, Assets/accounts affected, Simulation status, Risk categories, Required human explanation, Next action.
