# Security Policy

This repository is a documentation and agent-skill pack. It must not contain private keys, seed phrases, wallet mnemonics, production API keys, opaque binaries, or remote installers.

## Safety commitments

- No real private key or seed phrase examples.
- No remote shell execution in installer scripts.
- No binaries, minified blobs, or hidden payloads.
- No instructions that bypass wallet approvals, spending policies, custody controls, or user confirmations.
- Signing examples must use placeholders, local test validators, or managed signer abstractions.

Report unsafe guidance, broken routing, or outdated Solana references through a GitHub issue.
