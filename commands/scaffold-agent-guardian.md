# Command: scaffold-agent-guardian

Scaffold a safe transaction-authority boundary for a Solana AI agent.

## Procedure

Load `skill/SKILL.md`, `wallet-access-models.md`, `policy-engine.md`, `transaction-simulation.md`, and `audit-logs.md`. Infer or ask for network, signer type, funds at risk, approved programs, max spend, approval threshold. Generate architecture, file tree, policy manifest, signer boundary, simulation/risk flow, audit schema, emergency controls, and tests for ALLOW / APPROVE_REQUIRED / BLOCK / PAUSED.
